# 東京旅行 Web App

## 功能
- 行程表（可輸入）
- Google Maps
- 天氣資訊（OpenWeather API）
- 待辦清單（可新增刪除）
- 行李清單（預設分類）

## 部署到 GitHub Pages
1. 建立 GitHub Repository
2. 上傳這個專案的檔案
3. 在 GitHub 設定中啟用 Pages，選擇 main branch
4. 網站就會公開！

## 天氣 API 設定
- 前往 [OpenWeather](https://openweathermap.org/) 註冊並取得 API Key
- 在 `script.js` 中將 `YOUR_API_KEY` 替換成你的 API Key

## 手機適配
- 已使用 RWD 設計，適合手機瀏覽
