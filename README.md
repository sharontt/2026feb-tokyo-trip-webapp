# 東京旅行 Web App V2

## 功能
- 行程表：2月6日至10日，每天獨立頁面，時間線呈現，地點自動更新 Google Map
- 天氣：顯示當天天氣與預報，並提供穿衣建議
- 購物清單：分類（化妝品、食物、日用品），可新增刪除
- 行李清單：可勾選、可新增刪除、支援拖動排序
- 設計：現代前衛、透明感、莫蘭迪色調、RWD

## 部署到 GitHub Pages
1. 建立 GitHub Repository
2. 上傳檔案
3. 啟用 Pages → 選擇 main branch
4. 網站公開

## 天氣 API 設定
- 前往 [OpenWeather](https://openweathermap.org/) 註冊並取得 API Key
- 在 `script.js` 中替換 `YOUR_API_KEY`
